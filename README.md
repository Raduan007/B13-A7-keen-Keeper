# 📌 Keen Keeper
## Timeline & Check-In Tracker 📊

---

# 📖 Short Description

Keen Keeper is a simple and interactive web application that helps users track their daily interactions such as **calls, texts, and video check-ins** with friends.

It provides a clean dashboard with a **timeline view** and **visual statistics**, helping users understand their communication habits in a modern and user-friendly interface.

---

# 🛠️ Technologies Used

- ⚛️ React.js  
- 🚏 React Router DOM  
- 🎨 Tailwind CSS  
- 📊 Recharts  
- 🔔 React Toastify  
- ⚡ JavaScript (ES6+)  
- ⚡ Vite  

---

# ✨ Key Features

## 📅 Timeline Tracking

Track all your interactions (call, text, video) in a clean and filterable timeline view.

---

## 📊 Interactive Stats Dashboard

Visualize your activity using beautiful pie charts showing distribution of calls, texts, and videos.

---

## 👥 Quick Check-In System

Instantly log interactions with friends using one-click buttons and get real-time toast notifications.
